How I made this Website with only ChatGPT Prompts

Ground Rules

As part of doing this project I had to recognise a couple of aspects. My day job is being an engineer and I know how to research topics.
The result of this meant to get a true reflection of the capabilities of ChatGPT in regards to me, not only someone with literally 0 experience in web dev, but 0 experience with html, css or javascript (past making a couple of post request postman scripts), i had to make sure i limited myself only to ChatGPT, this meant no googling specific issues, no watching youtube videos, and no asking friends who are Web Developers themselves.

The End Goal

As with any project, goal, or plan, there needs to be something at the end where you can reflect and say you completed what you set out to do. Its so easy to constantly want to improve and just tweak this or that thing, and before you know it years have passed and you havnt shown anyone anything. So for me i set a couple of targets i wanted to achieve before publishing on LinkedIn.

  • The Website need a page for the post, but also a home page and a contact page
  • It needs to have a level of presentation (not winning any awards, but more than black text on a white screen)
  • The website needs to be navicable

Getting Started

The first prompt was the easiest:

"I want to make a website using CSS and JavaScript, write me a basic framework in JavaScript"

I was given a html and CSS file. Nothing extravegant but it created enough that it had some navigation buttons, a footer, and some placeholder text in the middle. Great start.
Or it would have been, but im no Web Dev, ive not really even used Visual Studio Code before, so a small issue to fix was next:

"how do I add javascript in visual studio code"

Easy enough to get an answer to. As I now have a very basic repo of a website, I wanted to take a look at it:

"cool, ive added the basic css html and javascript files to a visual studio code project, what do I need to do to build the project and host it on a localhost port?"

I then got the relevant information of how to intall npm and the commands needed to deploy locally. I had a small issue with npm but I was able to past the error and get options to remediate (in this case, restart Visual Studio Code).
I tried a couple of other things to test how ChatGPT works, like asking it to repeat a response from further up the conversation. It did this no issue, perminance of the chat is one of the best aspects of ChatGPT especially in projects like this. Way later in the process I was concerned at what would happen when I went to bed, how would I get it to know what work had already been done, to get back to the same place being able to ask the same questions? Its not an issue, the chat perminance is perminant, im not going to test it, but as long as you dont actively exit out, the chat remains (you can also have more than 1 chat if you have mutiple projects).
At this point I had a very basic locally hosted Website. With no experience in this area of development, this was achived in about 30 mins (I was also watching a twitch stream).
Inital take away? If you've done something similar even a single time before youd probably get to the same place in about 10 minutes, if youre experienced, youd probably not only do it faster, but also have set up things for the future, like folder structures etc, this isnt going to replace Comp Sci, or having a mentor yet. But not having that available, this is great.
Does this opinon change if we want to do something a bit more challenging?

Creating a new page

Time to increase the output, one of my goals was to have a page for this blog, so at some point I was going to have to find out how that was done, but I didnt really want to do any thinking.

"in this project we have a value of Contact which has references around the project as being intended as a button to go to a new page, how do I both create this page and ensure it links when I click that button?"

I thought it would be enought to essentially be given the entirely of the code, but it gave me instructions on how to create a new page instead. Combining this with a follow up asking for basic html for that page meant I went from having no idea to having it done in about 30 seconds.
This general process continued till i had a header image, a few pages, and buttons that do things when you hover over them.

Summary

You've probably seen those posts on LinkedIn, "the 10 AI tools you need to learn now to not fall behind", "If you arent using ChatGPT everyone will pass you".
Has it been useful, sure. Are there probably a lot of things I could do with it that I havnt even thought of, of course. But currently, I dont feel anyone is going to be hired because they can prompt ChatGPT well, nothing done here couldnt have been done by an experienced engineer in far less time, or by watching the right youtube video. Further to this, when you get to the complex problems, the massive repos of hundreds of files and hundreds of thousends of lines of code, you cant just paste them in to the text box like I did and say "tell me what the issue is".
The real reason I used ChatGPT for this is becuase when I tried to search for this on youtube the only videos I got were about why I should make a website, and which free vs paid builder sites I can use. ChatGPT is what you imagine Google is before you actually press search and get 17 advertisments, and the entire front page being tailored by companies who have precisely and explicitly defined their content expressly to hit the first page for what youve searched. I didnt want to use one of these sites but it does allow a great segway to todays sponsor, Squarespace.com, wait, this isnt youtube. But with more ease you could create the same thing that would look much better.
So what are your goals? If its to do something specific like making a website, thousends of people have already thought of the best way to make you do the least work to achieve that, but if its to learn how to do it, ChatGPT has been pretty good. I didnt put all the interactions as this is long enough as it is, but frequently if I have a problem it would give multiple suggestions of how to fix, if I wanted to do somethign it would give me multiple ways to achieve it.
And I think for me, most importantly, its given me confidence to just give stuff a go, a major hurdle is not knowing where to start, this is perfect for that. Dont become reliant on it, but learn how it works and get the most out of it.
